### О проекте

Данный репозиторий содержит исходный код официального информационного ресурса Западного Архангельска. Портал создан для информирования населения, стандартизации документооборота и хранения ключевых сведений о структуре Директората.

Сайт представляет собой **вики-справочник** по ролевой игре в сеттинге постапокалипсиса. 

**Предыстория мира:**
> 1 апреля 2032 года глобальный кризис, вызванный кибератакой, привел к краху современной цивилизации. Государства распались. В Западном Архангельске власть перешла к Директорату — союзу технических специалистов и руководителей критической инфраструктуры.

---

### Структура портала

Сайт разделен на ключевые разделы для удобства навигации:

*   **Терминология** — словарь принятых сокращений (БЕ, ЗА) и понятий.
*   **История** — хроника глобального кризиса и формирования Директората.
*   **Директорат** — структура Координационного Центра, должности и обязанности.
*   **География** — карта региона, промышленность, энергетика и сырьевая база.
*   **Документация** — классификация документов (Законы, Указы, Приказы) и иерархия силы.

### Технические детали

Проект разработан для размещения на **GitHub Pages**.

*   **HTML5** — семантическая разметка.
*   **CSS3** — стилизация без использования фреймворков.
    *   Переменные CSS для цветовой схемы.
    *   Адаптивная верстка (Flexbox/Grid).
    *   Индустриальный минимализм.

### Установка и запуск

Проект не требует сборки. Для локального запуска:
1. Склонируйте репозиторий.
2. Откройте файл `index.html` в браузере.

Или просто перейдите по ссылке GitHub Pages.
